
学习一门编程语言唯二方法：
1.选择一套优秀的学习资料，可以是书和视频，重点是从头尾学习下来，跟着书本介绍，把代码录入一遍。
2.实践练习，找下实战项目，实践下来，之后就是日益精进了。

因为计算机语言本身的特性只是它的语法。而计算机语言的语法无外乎是表达以下几个方面的概念:
一. 数据的表达
1.数据类型
2.值的储存：变量、常量、字面量

二、流程（控制流）的表达
1.顺序语句
2.分支语句 if switch while
3.循环语句 for

三、代码复用的封装
1.函数式编程：函数的定义和调用
2.面向对象编程：类的定义和调用
3.命名空间的定义和调用：库，模块

四、熟悉 api
1.语言常用标准库的api

而当对这些概念有了基本的印象,并学会了用一门计算机语言表达这些概念之后，接下来对于初学者来说最为重要的应该就是三件事情：
1.编程范式
2.设计模式
3.基本的数据结构

五、常见的编程范式有：
1.面向过程的范式：这个应该是对于初学者而言最为容易理解的方式，几乎所有的主流语言都支持这一范式，初学者应当首先学会这种范式
2.面向对象的范式：这个应当是最为主流的编程范式，从软件开发的角度看， 它的优点在于能够用面向对象的方式贯穿软件开发的整个生命周期，包括软件的设计、编码、测试函数式编程，在并行计算的时代
3.函数式编程的范式：将会成为另一大主流，推荐对其做一些了解。

五、数据结构和算法
算法是计算机科学的灵魂，它的意义在于让你理解计算机的工作方式的本质。对于初学者而言，不需要掌握过于复杂的算法，但是应当熟练地掌握
1.基本的排序、检索、遍历的算法。
2.数据结构亦是类似，初期掌握数组，链表，栈，队列，哈希表，栈即可。

算法，有人看到「算法」两个字可能就晕了，觉得很深奥，其实并不是。算法就解决一个具体问题的一系列步骤，其实并不难懂，不要被它的表象给吓到哦~
比如说解决如何把大象装进冰箱的问题，分两步，第一步打开冰箱，第二步把大象塞进去。这就是这个问题的算法。

六、技术人成长，应该专注于哪些 “不变” 的底层知识？
1.计算机原理：基本的硬件知识，知道 cpu 内存 硬盘的原理，及之前通信的原理
2.操作系统：linux shell
3.网络知识：七层模型、网络协议
4.数据库：sql
5.数据结构和算法
6.设计模式
7.编程范式
8.运行时 js-浏览器 / node，java-jvm
